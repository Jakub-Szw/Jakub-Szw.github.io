# Personal Portfolio Website

This repository contains the source code for my personal website, built primarily with HTML and CSS. For code clarity, I used JavaScript and JSON, and I deployed it to GitHub Pages. I will continue to use this site to showcase my projects, contact information, and the opportunities I have taken part in.
## Live Demo

[jakub-szw.github.io](https://jakub-szw.github.io/)


## Features

- Responsive Design: It's fully optimized for mobile, tablet, and desktop views.
- Blog Post: Include a Blog category where my Blogs about opportunities will be posted.


## Installation

To run the website on your local machine, simply clone the repository and use the VSCode Live Server extension to deploy it.
